# Finis Trabaja - Complementos de Ingeniería de Software

Este proyecto consiste en una plataforma web local para la gestión de ofertas laborales y vinculación estudiantil de la Universidad Finis Terrae.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript moderno
